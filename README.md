# Semantic Portfolio Website

## Project Description

This project is a multi-page personal portfolio website developed using semantic HTML5 elements and accessibility best practices following WCAG guidelines.

## Features

* Semantic HTML5 structure
* ARIA labels and roles
* SEO-friendly meta tags
* Accessible contact form
* Keyboard navigation support
* Screen reader compatibility
* Responsive design

## Pages

* Home
* About
* Projects
* Contact

## Technologies Used

* HTML5
* CSS3

## Accessibility Features

* Semantic landmarks
* Skip navigation link
* ARIA attributes
* Alt text for images
* Focus indicators
* Proper heading hierarchy
